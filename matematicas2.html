Polinomios


multiplicación de polinomios


(2x + 3)(4x - 5)
Para multiplicar estos polinomios, debes distribuir cada término del primer polinomio sobre el segundo polinomio y luego simplificar:
2x * 4x + 2x * (-5) + 3 * 4x + 3 * (-5)
8x^2 - 10x + 12x - 15
Simplificando la expresión, obtenemos:
8x^2 + 2x - 15

suma de Polinomios


(3x^2 + 5x + 2) + (2x^2 + 4x + 1)
Para sumar los polinomios, simplemente sumamos los términos correspondientes:
(3x^2 + 5x + 2) + (2x^2 + 4x + 1) = (3x^2 + 2x^2) + (5x + 4x) + (2 + 1)
Simplificamos los términos semejantes:
5x^2 + 9x + 3
Por lo tanto, la suma de los polinomios es 5x^2 + 9x + 3.

resta de polinomios


(3x^2 + 4x + 2) - (2x^2 - 3x + 5)
Para restar estos polinomios, debes cambiar el signo de todos los términos del segundo polinomio y luego combinar los términos semejantes:
3x^2 + 4x + 2 - 2x^2 + 3x - 5
(3x^2 - 2x^2) + (4x + 3x) + (2 - 5)
x^2 + 7x - 3
La resta de los polinomios resulta en x^2 + 7x - 3.

división de polinomios


Dividir el polinomio (2x^3 + 5x^2 - 3x + 1) entre (x - 2).
1. Coloca el dividendo y el divisor en la división larga:
(x - 2) | 2x^3 + 5x^2 - 3x + 1
2. Divide el primer término del dividendo (2x^3) entre el primer término del divisor (x):
(x - 2) | 2x^3 + 5x^2 - 3x + 1 - (2x^3 - 4x^2)
3. Realiza la resta y lleva el siguiente término (-4x^2) a la siguiente línea:
(x - 2) | 2x^3 + 5x^2 - 3x + 1 - (2x^3 - 4x^2) ________ 9x^2 - 3x
4. Divide el primer término de la nueva línea (9x^2) entre el primer término del divisor (x):
(x - 2) | 2x^3 + 5x^2 - 3x + 1 - (2x^3 - 4x^2) ________ 9x^2 - 3x - (9x^2 -18x)
5. Realiza la resta y lleva el siguiente término (-18x) a la siguiente línea: ________
(x - 2) | 2x^3 + 5x^2 - 3x + 1 - (2x^3 - 4x^2) ________ 9x^2 - 3x - (9x^2 -18x) ________ 15x + 1
6. Divide el primer término de la nueva línea (15x) entre el primer término del divisor (x): ________
(x - 2) | 2x^3 + 5x^2 - 3x + 1 - (2x^3 - 4x^2) ________ 9x^2 - 3x - (9x^2 -18x) ________ 15x + 1 - (15x - 30)
7. Realiza la resta y lleva el siguiente término (-30) a la siguiente línea: ________
(x - 2) | 2x^3 + 5x^2 - 3x + 1 - (2x^3 - 4x^2) ________ 9x^2 - 3x - (9x^2 -18x) ________ 15x + 1 - (15x - 30) ________ 31
El resultado de la división es: (2x^3 + 5x^2 - 3x + 1) / (x - 2) = (9x^2 - 3x) + (15x + 1) / (x - 2) con un residuo de 31.
